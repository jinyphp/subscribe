# 010. Admin Interface Layout - TDD Implementation

## 개요
jiny/admin 패키지 기반 관리자 인터페이스 레이아웃 및 사이드바 통합

### jiny/admin 패키지 활용
- **기존 레이아웃 확장**: 현재 Jiny Admin 사이드바에 구독 관리 섹션 추가
- **일관된 UI/UX**: 기존 관리자 인터페이스와 동일한 디자인 시스템
- **권한 통합**: 기존 admin 미들웨어 및 권한 시스템 활용
